1. Heuristic questions to ask from this Titanic data, and key question to research¶

This Titanic data may disclose correlations from multiple aspects, strong or weak, some are obvious but less interesting (e.g. Pclass and Fare may have very strong correlation, but little in value of research), while others are less obvious but very valuable to find out, one of this question is:

    What features (column variables) may made the passenger more likely to survive?

As of this time, we are not exploring correlations between a result (Y) and multiple independent variables (X1, ..., Xn), we will do that in machine learning project. So here we deduce the goal of this project to:

    What are the correlations between two variables - 'Survived' and one another variable?
    How do we present and calculate these correlations?
    Which one is the strongest, etc.?

Throughout this project, I will use Pandas and Numpy to find solutions to the most interested questions, and create a report to clearly demonstrate my findings. Since it is an open-ended project, there may be more than one good answers exist, what I believe is that my finding should be tentatively a good answer.

1.1 Questions of what features may made the passenger more likely to survive¶

Looked at the data, assuming a passenger's survivability was purely due to availability of lifeboat from arrangement in order, we have following questions to find true interesting answers via data analysis (hypothesis to validate):

    1. Would passengers of higher 'Pclass' were more likely 'Survived'?
       Hypothesis: Yes, since higher class passengers were more likely obtaining better services.
    2. Would passengers paid higher 'Fare' were more likely 'Survived'?
       Hypothesis: Yes, passengers paid higher fare were more likely obtaining higher priorities in service.
    3. Would passengers of younger 'Age' were more likely 'Survived'?
       Hypothesis: Yes, since youth or younger passengers were more likely given to survive in crisis.
    4. Would passengers of female in 'Sex' were more likely 'survived'?
       Hypothesis: Yes, since female passengers were more likely given to survive in crisis.
    5. Would passengers with higher 'Parch' were more likely 'Survived'?
       Hypothesis: Yes, they were more likely to obtain lifeboat space,
                   either from their parents' give-away or been survived to take care of survived child.
    6. Would passengers with higher 'SibSp' were more likely 'Survived'?
       Hypothesis: No, they were more likely to give up their opportunities to their 'SibSp'.

1.2 Method to use for data analysis¶

Key of the analysis is to use Pandas and Numpy to calculate Pearson product-moment correlation coefficient (Pearson's r) between 'Survived' and other variables from the above questions.

Note: variable - 'Sex' isn't fit to Pearson's r calculation, we should transfer it to a scalar variable in each row.

1.4 Compute the Pearson product-moment correlation coefficent between 'Survived' and other variables¶

First come up with an utility or library function to compute Pearson's r¶
In [4]:
# NumPy's corrcoef() function can be used to calculate Pearson's r, also known as the correlation coefficient.
# But here, I'd rather use an own version of Pearson's r calculation using Vectorized operations.
# Also confirmed that it produces same value as np.corrcoef()

# x, y are Pandas Series, data frame column, or numpy array
def correlation(x, y):
    '''
    Fill in this function to compute the correlation between the two
    input variables. Each input is either a NumPy array or a Pandas
    Series.
    
    correlation = average of (x in standard units) times (y in standard units)
    
    Remember to pass the argument "ddof=0" to the Pandas std() function!
    '''
    std_x = (x - x.mean()) / x.std(ddof=0)
    std_y = (y - y.mean()) / y.std(ddof=0)
    
    return (std_x * std_y).mean()

In all statistical analysis below, I used:

- Null Hypothesis (H0): the correlation between two investigated variables is insignificant (negligible).
  Alternative Hypothesis (Ha): Otherwise.
- Significance level criteria:  = 0.01 (or 1%)
- Samples' size:      N =  714 (DF = N-1 = 713)
- To calculate p-value significance (2-tailed) from Pearson's r, used : www.graphpad.com/quickcalcs/pValue1/
